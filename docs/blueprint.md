# **App Name**: Customer Meet 2026 Check-in

## Core Features:

- Secure Team Authentication: Implement Firebase Email/Password authentication to allow only authorized team members to log in and manage their session.
- Firebase Project Setup & Rules: Initialize a Firebase project with Firestore, Authentication, Cloud Storage, and Functions. Define Firestore collections for 'guests', 'teamUsers', and 'checkInLogs' with the specified schemas and configure security rules to restrict access to authenticated team members only.
- Team Login Interface: Develop a dedicated login page featuring the 'Customer Meet 2026' app logo and text, email and password input fields, a login button, and robust error handling for incorrect credentials with friendly messages.
- Authenticated Dashboard Navigation: Automatically redirect team members to a dashboard home page upon successful login. Implement a logout button that is consistently visible across all authenticated pages.
- Mobile-First Responsive Design: Ensure the entire application interface, including login and dashboard views, is fully responsive and optimized for seamless use on various screen sizes, including phones, tablets, and desktops.

## Style Guidelines:

- The primary color for interactive elements and branding is a professional mid-tone blue (#2273C3), evoking trust and clarity.
- A very light, almost white, background color (#F0F2F4) is used to maintain a clean, spacious feel, subtly tinted with the primary hue for coherence.
- An energetic cyan-aqua accent color (#1AE6E6) will be used sparingly for calls-to-action and important highlights to create clear contrast and draw attention.
- The 'Inter' sans-serif font is recommended for all text, providing a clean, modern, and highly readable appearance suitable for both headlines and body content.
- A clean and professional layout prioritizing readability and ease of navigation on all devices, with thoughtful spacing and intuitive element placement.
- Utilize modern, simple line icons that complement the clean design and clearly convey actions and information.
